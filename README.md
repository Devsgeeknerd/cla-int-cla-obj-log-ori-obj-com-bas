# Introdução

***Conteúdo da Aula:***

Uma das premissas das linguagens orientadas a objetos é a possibilidade de representar coisas que nós temos no mundo real com estruturas de código.

A vantagem dessa abordagem é que o código fica muito mais intuitivo e claro, já que na maioria das vezes nós teremos um código escrito que fará completo sentido para qualquer pessoa que o ler, já que ele terá relação direta com o mundo real.

A maneira como nós fazemos essa representação do mundo real dentro do código a ser desenvolvido é feita primariamente através de classes e seus respectivos objetos.
